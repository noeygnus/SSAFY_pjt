# 📈 댓글 증강 시스템 프로젝트 (데이터 크롤링 프로젝트)

Django 기반 웹 애플리케이션에서 Selenium을 활용한 실시간 데이터 크롤링과 LLM 기반 데이터 증강 파이프라인을 구현하는 것을 목표로 합니다.
사용자가 입력한 회사명을 기준으로 토스증권 커뮤니티 댓글 데이터를 실시간 수집하고, 이를 전처리 및 증강하여 통합 결과를 웹 화면에 제공합니다.

* 데이터 전처리 기준 
: 너무 짧은 댓글, 특수 문자 도배, ㅋㅋㅋㅋㅋㅋ 같은 반복 패턴, 욕설/혐오/비방 댓글 

핵심 목표
Django 기반 웹 서비스 구조 이해 및 구현
Selenium을 통한 동적 페이지 데이터 크롤링
댓글 데이터 전처리 및 품질 개선
GMS LLM API 기반 텍스트 증강
원본, 정제, 증강 데이터 통합 출력
예외 처리 및 사용자 경험 개선

# 3. 시스템 구조
사용자 입력
   ↓
회사명 검증
   ↓
토스증권 회사 검색
   ↓
최상단 회사 자동 선택
   ↓
커뮤니티 댓글 20개 크롤링
   ↓
데이터 전처리
(길이 필터링 / 특수문자 제거 / 부적절 댓글 제거)
   ↓
LLM 기반 댓글 증강
   ↓
데이터 통합
   ↓
결과 출력 + 요약 + 예외처리


# 4. 구동방법
① 가상환경 생성
python -m venv venv
venv\Scripts\activate

② 필수 라이브러리 설치
pip install -r requirements.txt

③ DB 마이그레이션
python manage.py makemigrations
python manage.py migrate

④ 서버 실행
python manage.py runserver

# 5. 느낀점
이번 프로젝트를 진행하면서 가장 크게 느낀점은 수집한 데이터를 얼마나 의미있게 정제하고, 사용자에게 가치있는 형태로 제공할 수 있는가에 대한 중요함이었습니다.
특히 페이지 구조를 분석하고 로딩 시간을 고려하고 원하는 요소를 찾기 위해 여러번 오류를 수정하면서 실제 서비스 개발은 예상보다 훨씬 많은 변수와 예외 상황을 고려해야 한다는 점을 직접 체감했습니다.
또한 SSAFY에서 제공된 기본 스켈레톤 코드를 가지고, 단순히 AI를 부려먹는것이 아니라 똑똑하게 활용한다면 단순 구현자가 아닌 데이터를 더 의미있게 만들 수 있다는 것을 느꼈습니다.
데이터 정제와 AI 증강까지 직접 구현해야 한다는 점이 어렵고 막막했지만, 완성하고 나니 Django, AI, Selenium 을 하나로 연결했다는 점에서 성장한 프로젝트였습니다.
